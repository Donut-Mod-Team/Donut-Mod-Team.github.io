The Donut Mod is an extension addon for the TIBCO Spotfire platform, which complements its environment by allowing the users to visualize their data using this donut chart representation.


Getting Started

On the Getting Started page you will be able to find the prerequisites and an installation guide for the mod. This will teach you how to add the mod to Spotfire, explain the settings and features, as well as a nifty tutorial video on how to use the mod!


About The Mod

The }}">About The Mod page will tell you all about the mod itself, such as it's features, as well as a dedicated section for developers that want to contribute to or build upon the mod. If you are interested in knowing more about the frameworks and different aspects of the mod itself, rather that simply what it does or using it, this is the place to go!


About Us

About Us is the page where you can get to know us, the developers! Here you can see some pictures of us, as well as the primary roles we have been in charge of throughout the development of this mod. Are you curious to know more about us? Come say hi!


Download and Release

To download the mod and see all former, and upcoming, releases, head over to the Download & Releases page. Here you will find the information you need to get the Donut Mod all for yourself.
